Contains all the packages and their source code for this command line tool. Every package corresponds to a namespace.

#### Common - `CommonNs`
* `C++11`
#### Core - `CoreNs`
* `C++11`
#### Fan - `FanNs`
* `C++11`
#### Interface - `IntfNs`
* `C++11`, `Lex`, `Yacc`
